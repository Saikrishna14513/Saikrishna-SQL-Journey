# Saikrishna-SQL-Journey

**My SQL Learning Journey**

This repository documents my journey of learning SQL for Data Analyst roles.  
It includes key concepts, definitions, and practical queries.

**SQL Definition:**

SQL Stands for Structured Query Language. SQL is a programming language used to communicate with databases. By using SQL queries, we can store, process, analyze, and manipulate databases.

**Topics Covered**
### 1. Basic Queries**

- **SELECT**: Select is used to retrieve data from the table.
- **WHERE**: Where is used to filter records based on conditions  
- **ORDER BY**: Order by is used to sort the results of a returned query based on an ascending or Descending order. 
- **LIMIT**: Limit is used to restrict the number of rows of a returned query. 


### 2. Joins
Joins are used to combine data from multiple tables.

- **INNER JOIN**: 
- **LEFT JOIN**: Returns all records from the left table and matching records from  the right table.
- **RIGHT JOIN**: Returns all records from the right table and matching records from the left table.  
- **FULL JOIN**: Returns all records from both tables. 


### 3. Aggregations
Aggregation functions perform calculations on multiple rows.

- **COUNT()**: Counts rows  
- **SUM()**: Calculates total  
- **AVG()**: Finds average  
- **MAX() / MIN()**: Finds highest/lowest value  
- **GROUP BY**: Groups rows for aggregation  


### 4. Subqueries
A subquery is a query inside another query.

- Used for filtering, calculations, and comparisons  
- Can be used in SELECT, WHERE, or FROM  


### 5. Window Functions
Window functions perform calculations across a set of rows without grouping rows.

- **RANK()**: Assigns rank to rows  
- **ROW_NUMBER()**: Assigns unique row numbers  
- **PARTITION BY**: Divides data into groups  

---

### 6. CASE Statements
Used for conditional logic in SQL (like IF-ELSE).

- Helps categorize or transform data
